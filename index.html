🍎 macOS / 🐧 Linux

1. Klicke unten auf Skript (.sh) laden.
2. Lege die Datei neben deinen Bilder-Ordner.
3. Öffne das Terminal im Ordner und gib ein:
bash verschiebe_bilder.sh
4. Alle Bilder werden sofort verschoben!
